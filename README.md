# 🚀 Instagram to YouTube Automation Pipeline

A fully automated content pipeline built with n8n, Python, FFmpeg and Railway!

## ✅ Features
- Download Instagram reels via Telegram bot
- Add animated title + watermark using FFmpeg
- Schedule YouTube uploads at 6 PM daily
- 100% cloud hosted on Railway

## ⚙️ Tech Stack
- n8n — Workflow automation
- Python + Flask — Backend API
- FFmpeg — Video processing
- Railway — Cloud hosting
- Telegram Bot API
- YouTube Data API v3
- Instaloader

## 🛠️ Setup
1. Clone this repo
2. Deploy to Railway
3. Set environment variables
4. Import n8n workflow
5. Connect Telegram bot

## 📌 Environment Variables
INSTAGRAM_USER=your_username
INSTAGRAM_PASS=your_password
